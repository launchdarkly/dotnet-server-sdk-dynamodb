Remarks

This object can be modified with DynamoDBDataStoreBuilder methods for any desired custom DynamoDB options. Then, pass it to LaunchDarkly.Sdk.Server.Components.PersistentDataStore(LaunchDarkly.Sdk.Server.Interfaces.IPersistentDataStoreAsyncFactory) and set any desired caching options. Finally, pass the result to LaunchDarkly.Sdk.Server.ConfigurationBuilder.DataStore(LaunchDarkly.Sdk.Server.Interfaces.IDataStoreFactory).

Examples
    var config = Configuration.Builder("sdk-key")
        .DataStore(
            Components.PersistentDataStore(
                DynamoDB.DataStore("table-name")
            ).CacheSeconds(15)
        )
        .Build();
